# Contributing to UniAI Kernel

Thank you for your interest in contributing! We welcome contributions from everyone.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Install dependencies using `uv` (recommended):
   ```bash
   uv sync
   ```
   Or using pip:
   ```bash
   pip install -r requirements.txt
   ```
4. Create a new branch for your feature or bugfix:
   ```bash
   git checkout -b feature/your-feature-name
   ```

## Development Workflow

1. Make your changes and write tests if applicable.
2. Ensure all tests and linting pass.
3. Commit your changes following conventional commits format.
4. Push your branch to GitHub.
5. Submit a Pull Request.

## Code Style

- Please follow standard Python PEP 8 guidelines.
- We use type hints for all functions and internal methods.

Thank you for helping make UniAI Kernel better!
